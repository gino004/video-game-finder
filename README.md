# 🎮 Video Game Finder

Video Game Finder is a web application that allows users to search for video games, explore detailed information, watch trailers, and manage a list of favorite games.

This project demonstrates the use of modern web development techniques, including API integration, dynamic content rendering, and client-side data storage.

---

## 🚀 Features

- 🔍 Search for video games by name  
- 🎮 Display game details such as image, rating, and release date  
- ▶️ Watch game trailers using YouTube integration  
- ❤️ Add games to a favorites list  
- ❌ Remove games from favorites  
- 💾 Favorites are saved using localStorage  

---

## 🛠️ Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling, layout, and animations  
- **JavaScript** – Application logic and interactivity  
- **RAWG API** – Fetching video game data  
- **YouTube API** – Retrieving trailers  
- **LocalStorage** – Saving user favorites  

---

## ⚙️ How It Works

- The user enters a game name in the search bar  
- JavaScript sends a request to the RAWG API  
- The response (JSON data) is processed and displayed dynamically  
- When a game is selected, additional details and a trailer are shown  
- Users can save or remove games from favorites, which are stored locally
