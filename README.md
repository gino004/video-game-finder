# 🎮 Video Game Search Engine

This is a web application that allows users to search for video games, 
view detailed information, watch trailers, and save their favorite games.

## 🚀 Features

- Search for video games using an external API
- Display game details (image, rating, release date)
- View YouTube trailers
- Add and remove favorites
- Favorites are saved to local storage (localStorage)

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- RAWG API (game data)
- YouTube API (trailers)

## 📌 Notes

This project demonstrates the use of JavaScript, APIs, JSON data, DOM manipulation,
events, and local storage. This work is still under development but includes the basic
and main functionalities.
